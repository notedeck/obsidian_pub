## 構文

```
CREATE [PUBLIC] SYNONYM シノニム名 FOR オブジェクト名;
```

### 例1：プライベート・シノニムの作成

```
CREATE SYNONYM emp_syn FOR hr.employees;
```

## 例2：パブリック・シノニムの作成

```
CREATE PUBLIC SYNONYM emp_syn FOR hr.employees;
```

### シノニムの削除

```
DROP [PUBLIC] SYNONYM シノニム名;

```

### 例

```
DROP SYNONYM emp_syn;
DROP PUBLIC SYNONYM emp_syn;
```
